# 💼 Personal Portfolio Website

This is my personal developer portfolio website built with **React** and **Vite**. It highlights my skills, projects, and contact information in a clean, responsive layout.

## 🚀 Features

- ⚡️ Fast build and development with [Vite](https://vitejs.dev/)
- 🎨 Styled using [Tailwind CSS](https://tailwindcss.com/)
- 🎯 Smooth scrolling and clean UI design
- 📱 Fully responsive layout for all screen sizes
- 🔗 External links to GitHub, LinkedIn, and email
- 🧠 Built with functional components and React Hooks
